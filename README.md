# Dev Checkpoint Tracker

Enterprise-style Spring Boot + React application for tracking development checkpoints and generating professional Excel reports.

## 🚀 Features

- Environment-wise checkpoint tracking
- Dynamic development step management
- Testing status handling (Success / Fault / Exception)
- Excel export with:
  - Styled headers
  - Borders
  - Merged cells
  - Color-coded YES/NO statuses
- Responsive React UI
- REST API integration
- Docker-ready setup
